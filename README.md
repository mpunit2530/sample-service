# sample-service
This is a sample nodejs REST service that loads data from static json file.

## Features

- Express
- REST API

## Requirements

- [node & npm](https://nodejs.org/en/)

## Installation

- `clone sample-service.git`
- `cd sample-service`
- `npm install`
- `npm start`

### GET Routes

- visit http://localhost:3000
  - /sailings/cheapest
  - /sailings/fastest
  - /sailings/cheapest-leg


### .env

- PORT = 3300
